# Creator Stats App

Creator Stats App is a small developer application used to fetch and review creator-account statistics for owned or authorized accounts.

Its current purpose is to support internal analytics workflows for music and creator profiles. The app is intended for private development and testing use, including sandbox and authorized API integrations.

## What the app does

- connects to official platform APIs when authorized
- fetches account-level and content-level statistics
- stores snapshot data locally for private analysis
- supports digest-style summaries of creator performance
